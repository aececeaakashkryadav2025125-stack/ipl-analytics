# 🏏 IPL Win Predictor

A machine learning web app that predicts the winning probability of a team during an IPL match.

## 🚀 Features

* Real-time win probability prediction
* Based on match situation (score, wickets, balls left)
* Built using Random Forest model
* Interactive UI using Streamlit

## 🛠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

## 📂 Project Structure

* app.py → Streamlit app
* models/ → trained ML model
* data/ → dataset
* requirements.txt → dependencies

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
